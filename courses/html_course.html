Bab 1: Pengenalan HTML

Di bab ini, kita akan mempelajari fondasi dari setiap halaman web, yaitu HTML. Anda akan belajar cara menstrukturkan konten.

  • Struktur dasar dokumen HTML5
  • Elemen heading, paragraf, dan list
  • Menambahkan gambar dan link
  • Penggunaan tag semantik: `<header>`, `<main>`, `<footer>`

Contoh Kode: Struktur Dasar

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

Bab 2: Styling dengan CSS

Setelah membuat struktur, saatnya mempercantik halaman dengan CSS. Kita akan membahas selector, properti warna, font, dan box model.

  • Cara kerja CSS: inline, internal, dan external
  • Selector dasar (tag, class, id)
  • Mengubah warna, font, dan background
  • Konsep Box Model: padding, border, margin

Contoh Kode: Memberi Warna

/* style.css */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

h1 {
    color: #333;
    text-align: center;
}

Latihan Statis

Soal: Buat sebuah selector CSS untuk menargetkan semua elemen `<p>` yang memiliki class `highlight` dan ubah warnanya menjadi biru.

Jawaban: `p.highlight { color: blue; }`

Bab 3: CSS Layout Modern

Bab ini fokus pada teknik layouting yang esensial untuk membuat website responsif, yaitu Flexbox dan CSS Grid.

  • Pengenalan Flexbox untuk layout satu dimensi
  • Properti Flexbox: `display`, `flex-direction`, `justify-content`
  • Pengenalan CSS Grid untuk layout dua dimensi
  • Membuat grid responsif dengan `fr` unit dan `repeat()`

Contoh Kode: Layout Flexbox

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
